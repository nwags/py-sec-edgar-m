# Canonical User Testing Handoff

## Goal

Define when the comprehensive cross-application user-testing program should begin.

## Start gate

Comprehensive user testing should begin only after:
- Wave 7.2 implementation is complete,
- one real shared RC cycle has been executed,
- the central evidence bundle and signoff flow have worked end-to-end,
- rollback path has been verified,
- no blocking release-lifecycle incident remains open.

## Important rule

User testing is still:
- post-Wave-7.2 stabilization,
- pre-cleanup / pre-shim-retirement,
- not a replacement for package-side and consumer-repo validation.
