# Canonical User Testing Start Gate

## Goal

Define when the comprehensive cross-application user-testing program may begin.

## Start-gate recommendation

The user-testing program should start only after:
- Wave 7.1 implementation is complete,
- one package-side RC/stable execution workflow is operational,
- one shared RC has been validated across all four repos,
- evidence bundle and signoff process are working,
- rollback path has been verified,
- no blocking release-lifecycle incident remains open.

## User testing remains

- a post-Wave-7.1 stabilization gate,
- a pre-cleanup / pre-shim-retirement gate,
- not a replacement for package/repo validation.
